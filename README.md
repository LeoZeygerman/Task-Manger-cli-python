📋 Task Manager CLI
Консольное приложение для управления задачами, написанное на Python.
Проект позволяет создавать задачи, отслеживать дедлайны, изменять данные, удалять задачи и просматривать статистику выполнения.
 
⸻
 
🚀 Возможности
➕ Создание задач
📖 Просмотр всех задач
🔍 Поиск задачи по названию
✏️ Изменение данных задачи
🗑️ Удаление задачи
📊 Статистика выполнения задач
⏳ Отслеживание дедлайнов
⚠️ Определение просроченных задач
💾 Сохранение данных в JSON-файл
 
⸻
 
📝 Структура задачи
Каждая задача содержит:
Название
Описание
Приоритет
Дедлайн
Статус выполнения
Пример:
Название: Изучить PostgreSQL
Описание: Освоить основы SQL и базы данных
Приоритет: Высокий
Дедлайн: 2026-07-01
Статус: В процессе
 
⸻
 
📊 Статистика
Приложение отображает:
Общее количество задач
Количество завершённых задач
Количество активных задач
Количество просроченных задач
Процент выполнения задач
Пример:
Всего задач: 10
Завершено: 7
В процессе: 2
Просрочено: 1
Процент выполнения: 70%
